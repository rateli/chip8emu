# CHIP-8 Emulator
A simple CHIP-8 Emulator written in Scala for learning purposes.

## Running the program
```sh
$ sbt "run /path/to/rom"
```

## Key mappings
Keypad
| 1 | 2 | 3 | C |
|---|---|---|---|
| 4 | 5 | 6 | D |
| 7 | 8 | 9 | E |
| A | 0 | B | F |
